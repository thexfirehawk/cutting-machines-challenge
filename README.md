# cutting-machines-challenge
Cutting machines web application

## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/thexfirehawk/cutting-machines-challenge.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the project
   ```sh
   npm start
   ```
4. Addittionally you can run tests
   ```sh
   npm test
   ```
